## About

Vibe Alarm is designed to make mornings feel natural.

No stress. No noise. No unnecessary features.  
Just a clean interface and a gentle wake-up experience.

---

## Features

- Gentle wake-up experience  
- Soft sound-based alarms  
- Clean, minimal interface  
- Fast and lightweight

1. Download the APK from the button above  
2. Install it on your Android device  
3. Allow installation from unknown sources if needed

## Tech

- Kotlin — core application logic  
- XML — UI layouts  
- Android SDK — platform
